## Alterações da Sprint 2

Nesta sprint, o projeto recebeu as seguintes atualizações:

- Implementação da página de agendamentos (`Registros`) com cadastro, edição, exclusão, alteração de status e filtros.
- Persistência dos agendamentos no `localStorage`.
- Dashboard com resumo de totais, status e valor total.
- Página de serviços com lista de serviços ordenada por popularidade.
- Navegação por rotas entre `dashboard`, `servicos`, `registros`, `visagismo` e `sobre`.
- Estrutura geral configurada em `app.config.ts`, `app.routes.ts` e `app.component.ts`.

### Como iniciar o projeto - SUGERIDO PELA EQUIPE FUNF JUNGS

1. Abra a pasta `app` no cmd.
2. Execute `npm install`.
3. Execute `npm run start`.
4. Acesse o local que aplicação que o codigo vai dar.

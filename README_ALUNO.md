# Projeto Angular Modelo

Este pacote contém uma aplicação Angular pronta para estudo e adaptação.

## Como executar

1. Extraia a pasta do ZIP.
2. Evite executar dentro de Google Drive, OneDrive, Dropbox, pendrive ou pasta de rede.
3. Abra a pasta extraída.
4. Execute o arquivo:

```text
iniciar.bat
```

5. Acesse:

```text
http://localhost:4200
```

Não feche o terminal enquanto estiver usando a aplicação.

## Onde estudar o projeto

O arquivo principal de orientação é:

```text
README.md
```

Ele explica:

- o fluxo da aplicação;
- a função das pastas;
- os arquivos que devem ser abertos em aula;
- a ordem recomendada de leitura;
- quais arquivos modificar para adaptar o projeto.
